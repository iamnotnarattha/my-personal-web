Crystal x Blue Space

The Crystal x Blue space project is a campaign that bridges online and offline experiences through Line OA. The UX/UI was designed for a seamless and intuitive user experience, making it easy to watch videos, collect Crystal Points, submit receipts, and redeem rewards.

Role & Responsibilities

UI/UX Design, UI Development

Duration

Oct – Nov 2024

Tools & Technologies

Figma, HTML, Tailwind CSS

Low-fidelity Wireframe

These wireframes illustrate the early ideation of user journeys, focusing on clarity and flow before visual styling. Key flows include
• Video viewing & point collection
• Receipt submission
• Rewards redemption
• Each flow was tested for logic and clarity before moving to high-fidelity design.

Design System

A scalable design system was created to maintain visual consistency and speed up development. It includes
• Modular components
• Icon sets
• Typography rules
• Color palette with accessibility in mind

Sample UI

The final UI brings everything together: clean visuals, smooth interactions, and brand consistency.
