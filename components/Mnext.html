MNEXT

Introducing a seamless financial solution that makes topping up easy, spending convenient, and payments instant with just a scan. Enjoy a fast and hassle-free digital payment experience, now enhanced with E-Gift, a new feature that allows you to scan NFC from participating vouchers.

Role & Responsibilities

UI/UX Design, UI Development

Duration

Feb – Apr 2024

Tools & Technologies

Figma, Svelte, Tailwind CSS

Design System

To ensure consistency and scalability, I developed a comprehensive design system that includes
• Color Palette: Brand and base colors for cohesive visual identity
• Components: Reusable UI elements such as buttons, cards, and forms
• Code Snippet: Tailwind CSS theme configuration for streamlined development

Sample UI

The user interface was designed to
• Simplify top-ups and payments via mobile
• Provide a clear dashboard for transaction summaries
• Enable smooth access to E-Gift rewards with real-time status updates
• Reflect the brand’s modern and friendly identity through a clean, intuitive layout

Key Feature: E-Gift

This feature allows users to scan NFC-enabled vouchers from participating merchants to redeem digital rewards such as gift cards and special offers. It's fast, secure, and designed for effortless access to exclusive deals—perfect for retail environments and promotional campaigns.
